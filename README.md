# React Simple Calculator
This is a simple calculator build with React and Tailwind CSS for leaning purpose.

## Features 
- Basic 4 operations (+, -, *, /)
- Parentheses handling
- Error handling
- Backspace and Clear functionality

## Technologie used
- React
- Tailwind CSS
